## Installation

```sh
git clone git@github.com:moreishi/tg-nextjs.git
```

For production environments...

```sh
$ npm install
$ npm run dev
```

For ENV

```sh
DB_HOST=localhost
DB_USER=myuser
DB_PASS=mypassword
LARAVEL_URL="http://127.0.0.1:8000"
```
